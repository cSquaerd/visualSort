# visualSort
## Demonstration of Visualized Sorting Algorithms

Contains the following algorithms:
* Bogo Sort
* Bubble Sort
* Cocktail Sort
* Insertion Sort
* Selection Sort
	* Stable (normal)
	* Double (finds min. & max.)
	* Unstable (swaps directly)
* Merge Sort
	* O(1) Extra Space a.k.a. "In-Place"
	* O(n) Extra Space
* Heap Sort
* Quick Sort
